# TCP Echo Server and Client

## Command
```C
./TCPEchoServer4 5000                            // at server side
./TCPEchoClient4 127.0.0.1 "Echo this!" 5000     // at client side

```

## Result
**At client**
```sh
Echo this!
```

**At server**
```sh
Handling client 127.0.0.1/43568
Finish handling this client!
```
